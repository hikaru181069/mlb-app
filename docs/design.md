Design Principles

Philosophy

デザインは装飾ではない。

「新しい選手と出会う」という体験を、最も気持ちよく実現するための手段である。

⸻

UX Principles

Discovery First

最優先は選手との出会い。

すべての画面はDiscover体験を中心に設計する。

⸻

Casual First

初心者でも楽しめる。

専門知識は不要。

難しい数字より

「この選手好きかも」

を大切にする。

⸻

Progressive Disclosure

情報は段階的に見せる。

カードでは

- 写真
- プレースタイル
- 相性

だけ。

詳細画面では

- OPS
- xwOBA
- OAA
- Statcast

まで見られる。

⸻

Explain AI

推薦には理由を表示する。

例

長打力と走力があなたの好みに近い選手です。

AIはブラックボックスにしない。

⸻

UI Principles

Apple Musicのように

- 大きな写真
- 余白
- 滑らかなアニメーション
- シンプルな配色

を重視する。

⸻

Interaction

Discover画面では

- ❤️ Like
- ❌ Pass
- ⭐ Favorite

だけで操作できる。

操作は迷わせない。

⸻

Motion

アニメーションは派手さではなく、

気持ち良さを重視する。

- カードスワイプ
- フェード
- スケール
- ハプティクス（モバイル）

を積極的に使う。

⸻

Visual Keywords

- Minimal
- Friendly
- Modern
- Premium
- Clean
- Comfortable
- Dynamic
- Human
